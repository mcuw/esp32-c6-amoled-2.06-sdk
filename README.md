# ESP32-C6 AMOLED 2.06inch library

## Description

This is a library for the ESP32-C6 AMOLED 2.06inch touch watch from waveshare.

## Features 

- Uses Platformio
- Example code

## Prerequisites

- ESP32-C6 AMOLED 2.06inch touch watch

## Get Started

Uncomment a `src` example in `platform.ini` then flash your watch.
